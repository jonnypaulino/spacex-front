# Fullstack Challenge 🏅 Space X API

## Descrição
Este é um desafio para podermos ver suas habilidades como Fullstack Developer. Nesse desafio você deverá desenvolver um aplicativo para listar informações da API SpaceX-API.

## Tecnologias
- Linguagem: JavaScript
- Framework: React.js

## Instalação e Uso
1. Clone este repositório para o seu computador.
2. No diretório raiz do projeto, execute o comando `npm install` para instalar as dependências.
3. Depois de concluída a instalação, execute o comando `npm start` para iniciar o aplicativo.
4. Abra o navegador e acesse `http://localhost:3000` para visualizar o aplicativo.
